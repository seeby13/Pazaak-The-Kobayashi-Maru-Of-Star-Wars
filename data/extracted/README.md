# Extracted Data

This folder contains generated data products created by the gameplay extraction pipeline.

Typical outputs include:

- `pazaak_state_changes.csv`
- `pazaak_draw_events.csv`

These files are produced from gameplay recordings and can be regenerated at any time by running the extraction scripts.

For this reason, generated extraction outputs are not tracked by git.

The folder itself is included in the repository to document the expected project structure and workflow.
